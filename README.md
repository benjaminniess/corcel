# <a id="installing-corcel"></a> Installing Corcel

You need to use Composer to install Corcel into your project:

in composer.json
```
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/benjaminniess/corcel.git"
        },
    ],
```

then
```
composer require jgrossi/corcel
```
